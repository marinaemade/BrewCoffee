Brew Coffee | Premium Coffee Experience
A modern, responsive full-stack web application designed for a boutique coffee shop in Toronto. This project focuses on a "Luxury" aesthetic, featuring smooth transitions, a clean UI/UX, and a robust functional menu.

Live Features:
Responsive Navbar: A sticky navigation bar that adapts to all screen sizes for seamless browsing.
Dynamic Hero Section: A high-impact landing area introducing the "Brew Haven" brand.
About Section: Highlights the café's story, bean sourcing, and community vibe using a dual-column layout.
Featured Menu: A grid-based menu that displays coffee items (like the Vanilla Latte) with hover-zoom effects and a local asset management system.
Interactive Contact Us: A functional contact form paired with a live Google Map centered in Toronto.

Dark/Light Mode: A built-in theme controller allowing users to toggle between a "Luxury" dark theme and a clean light theme using DaisyUI.

Social Connectivity: Fully integrated footer with links to LinkedIn, Instagram, and Facebook.

Tech Stack:
Frontend: React.js (Vite)
Styling: Tailwind CSS & DaisyUI
Deployment: Vercel
